À propos

Bienvenue sur Annonces App, une plateforme web permettant aux utilisateurs de publier, consulter, modifier et supprimer des annonces d'objets à vendre.

Fonctionnalités

  • Consulter les annonces
  • Créer un compte
  • Se connecter
  • Ajouter une annonce
  • Modifier ses propres annonces
  • Supprimer ses propres annonces

Technologies utilisées

  • HTML
  • CSS
  • JavaScript
  • Firebase Authentication
  • Firebase Firestore
